# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/),
version 2.1. The canonical text — including the standards, scope, and
enforcement guidelines — lives at:

> https://www.contributor-covenant.org/version/2/1/code_of_conduct/

By participating in this project (issues, PRs, discussions), you agree to
abide by it.

## In our own words

We want `usta-cli` to be a useful, calm place to work on developer tooling.
Be kind to people you disagree with. Critique ideas, not contributors.
If something feels off — about a comment, a review, a pattern of behavior —
say so or report it.

## Reporting

For private reports of conduct issues, email **mo7amed.3bdalla7@gmail.com**.
Reports are confidential. Maintainers respond within seven days.

For *security* issues (vulnerabilities, exposed secrets, etc.), use
[`SECURITY.md`](./SECURITY.md) instead — that flow has a different SLA and
disclosure window.
