# `usta-core`

Pure domain types and rules for the [`usta`](https://crates.io/crates/usta)
project scaffolder. No I/O, no async — safe to use from any layer.

This crate is the foundation of a hexagonal workspace; see the
[`usta` repo](https://github.com/sunduq-ai/usta-cli) for the architecture
overview and the binary CLI.
