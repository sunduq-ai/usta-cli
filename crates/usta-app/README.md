# `usta-app`

Use cases for the [`usta`](https://crates.io/crates/usta) project
scaffolder — scaffold, extract, verify, update, add — wired only against
the trait-based [`usta-ports`](https://crates.io/crates/usta-ports). No
concrete adapters here; that's the binary's job.

See the [`usta` repo](https://github.com/sunduq-ai/usta-cli) for the
hexagonal architecture this crate sits in the middle of.
