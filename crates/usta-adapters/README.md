# `usta-adapters`

Concrete adapter implementations for the
[`usta`](https://crates.io/crates/usta) project scaffolder — local
filesystem, MiniJinja renderer, `ignore`-based scanner, `inquire`
prompts, and so on.

You probably don't depend on this directly unless you're building an
alternative `usta` binary or a third-party adapter. See the
[`usta` repo](https://github.com/sunduq-ai/usta-cli) for the hexagonal
architecture.
